🚀 Демо: Брендовый поиск HVAC оборудования

Интерактивная демонстрация системы поиска с перекрестным подбором по техническим параметрам

📋 Как пользоваться демо-каталогом

1. Выберите бренд
Переключайтесь между брендами (Daikin, Mitsubishi, LG) для просмотра их ассортимента
2. Выберите товары
Новинка! Кликайте по товарам для множественного выбора. Повторный клик снимает выбор
3. Переключение на аналоги
Главная фишка! При переходе на другой бренд показываются только найденные аналоги в списочном формате
4. Точное соответствие 1:1, 2:2, 3:3
Выбрали 2 товара → при переходе на другой бренд видите ровно 2 аналога с процентами совпадения
💡 Как работает перекрестный подбор
Кликайте по товарам для выбора. Выберете 1 товар — увидите 1 лучший аналог в других брендах. Выберете 2 товара — увидите 2 лучших аналога. Попробуйте!
